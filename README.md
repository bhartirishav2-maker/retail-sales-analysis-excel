# 📊 Retail Sales & Profit Analysis

## 📌 Overview

This project analyzes a retail dataset to identify key business insights.

## 🎯 Business Problem

High sales but low profitability due to discounting and loss-making products.

## 🔍 Key Insights

* Some categories are generating losses
* Discounting is reducing profit margins
* Profit varies by region and product

## 🚀 Recommendations

* Reduce excessive discounts
* Focus on high-profit products
* Improve pricing strategy
